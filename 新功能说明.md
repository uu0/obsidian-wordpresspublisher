# Obsidian WordPress 插件 - 新功能说明

## 项目概述

本项目基于 [obsidian-wordpress](https://github.com/devbean/obsidian-wordpress) 进行了深度重构和功能扩展，新增了以下核心功能：

## 🎯 新增功能

### 1. 页面发布支持
- ✅ 支持发布为 **文章 (Post)** 或 **页面 (Page)**
- ✅ 在发布界面可以选择内容类型
- ✅ 页面类型自动处理分类和标签的兼容性

### 2. 智能 Slug 生成
- ✅ **自动生成 Slug**：从标题自动生成 URL 别名
- ✅ **双模式支持**：
  - **拼音模式**：中文标题自动转换为拼音（例如："减脂食谱" → "jian-zhi-shi-pu"）
  - **AI 翻译模式**：使用 AI 将中文标题翻译成英文 Slug（例如："减脂食谱" → "weight-loss-recipes"）
- ✅ **手动编辑**：支持在发布界面手动修改自动生成的 Slug
- ✅ **智能清理**：自动清理特殊字符，确保 Slug 格式正确

### 3. 特色图片功能
支持三种方式选择特色图片：

#### 3.1 本地图片
- 从文件系统选择图片
- 从 Obsidian 笔记库中搜索并选择图片
- 支持常见图片格式（JPG、PNG、GIF、WebP）

#### 3.2 Unsplash 集成
- 通过关键词搜索 Unsplash 免费高质量图片
- 网格式图片浏览界面
- 一键下载并设置为特色图片
- 自动触发 Unsplash 下载统计（符合 API 要求）

#### 3.3 AI 图片生成
- 根据文章内容自动生成图片描述提示词
- 使用 DALL-E 等 AI 模型生成特色图片
- 支持手动编辑提示词
- 实时预览生成的图片

### 4. AI 服务集成

#### 4.1 双引擎配置
- **文字处理 AI**：用于生成摘要、翻译 Slug、优化内容
- **图片生成 AI**：用于生成特色图片

#### 4.2 支持的 AI 提供商
- **OpenAI**：GPT-3.5/GPT-4（文字）、DALL-E（图片）
- **Claude**：Claude 3 系列（文字）
- **兼容 API**：支持任何 OpenAI/Claude 兼容的 API 服务

#### 4.3 AI 功能
- 自动生成文章摘要
- 中文标题翻译成英文 Slug
- 根据文章内容生成图片描述
- AI 图片生成

#### 4.4 配置验证
- 保存配置时自动验证 API 连接
- 实时反馈配置状态
- 支持自定义 Base URL（适配国内镜像服务）

### 5. 现代化发布界面

#### 5.1 卡片式布局
- 左右分栏设计，信息层次清晰
- 左侧：基本设置（标题、Slug、类型、分类、状态等）
- 右侧：特色图片和高级选项

#### 5.2 视觉优化
- 圆角卡片设计
- 悬停阴影效果
- 平滑动画过渡
- 支持 macOS Liquid Glass 效果（毛玻璃背景）

#### 5.3 交互优化
- 实时 Slug 生成预览
- 特色图片缩略图预览
- AI 功能一键触发
- 清晰的按钮状态反馈

## 🔧 配置说明

### 1. Slug 生成配置

在插件设置中：
- **自动生成 Slug**：开启/关闭自动生成功能
- **Slug 生成模式**：选择"拼音转换"或"AI 翻译"

### 2. Unsplash 配置

1. 访问 [Unsplash Developers](https://unsplash.com/developers)
2. 创建应用获取 Access Key
3. 在插件设置中输入 Access Key
4. 点击"验证"按钮测试连接

### 3. AI 配置

#### 文字处理 AI
```
AI 提供商: OpenAI / Claude
Base URL: https://api.openai.com/v1
API Key: sk-...
模型名称: gpt-3.5-turbo / claude-3-sonnet-20240229
```

#### 图片生成 AI
```
AI 提供商: OpenAI (DALL-E)
Base URL: https://api.openai.com/v1
API Key: sk-...
模型名称: dall-e-3
```

**提示**：
- 可以使用国内镜像服务的 Base URL
- 保存后点击"验证连接"确保配置正确
- 文字和图片 AI 可以使用不同的配置

## 🎨 使用流程

### 发布文章完整流程

1. **编写文章**：在 Obsidian 中编写 Markdown 文档

2. **打开发布界面**：
   - 点击插件图标
   - 或使用命令面板

3. **填写基本信息**：
   - 输入标题（自动生成 Slug）
   - 选择类型（文章/页面）
   - 选择分类
   - 设置发布状态

4. **选择特色图片**（可选）：
   - 点击"选择图片"按钮
   - 选择来源：本地/Unsplash/AI
   - 预览并确认

5. **高级选项**（可选）：
   - 手动输入摘要或使用 AI 生成
   - 添加标签

6. **发布**：
   - 点击"发布"按钮
   - 等待上传完成

1. **API 费用**：
   - Unsplash API 免费但有请求限制

2. **网络要求**：
   - Unsplash 和 AI 功能需要网络连接
   - 建议配置稳定的网络环境
   - 可使用国内镜像服务

3. **隐私安全**：
   - API Key 加密存储

   - 建议定期更换 API Key
