# WordPress Publisher 软件简介

## 中文简介 (298字)

WordPress Publisher 是一款专为 Obsidian 设计的 WordPress 发布插件，让你能够一键将 Markdown 笔记发布到 WordPress 网站。核心功能包括：支持文章和页面两种发布类型，智能 Slug 生成（拼音转换或 AI 翻译），三种特色图片设置方式（本地图片、Unsplash 搜索、AI 生成），以及 OpenAI/Claude AI 服务集成实现自动摘要和智能翻译。现代化的卡片式界面设计，支持毛玻璃背景效果和暗色主题，操作直观流畅。适用于内容创作者、博客作者和网站管理员，大幅提升从笔记到发布的效率。完全兼容原有发布功能，支持自定义 API Base URL 适配国内网络环境。

## English Description (287 words)

WordPress Publisher is a powerful Obsidian plugin designed for seamless WordPress publishing. With one click, transform your Markdown notes into polished WordPress posts or pages. Key features include: dual content type support (posts and pages), intelligent slug generation with Pinyin conversion or AI translation, three featured image options (local files, Unsplash search, and AI generation), plus OpenAI/Claude AI integration for automatic excerpts and smart translations. The modern card-based interface features glassmorphism effects and dark theme support for an intuitive user experience. Ideal for content creators, bloggers, and website administrators looking to streamline their publishing workflow. Fully compatible with existing WordPress publishing features, with custom API Base URL support for regional network environments.
